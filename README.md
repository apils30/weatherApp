# Weather app

This is a Single Page Application that retieves current weather information from Open Weather App by API.

## Tech-stack

* React
* Semantic-UI


## Getting started

```bash
npm install
```
OR

```bash
npm i
```
Once the installation process has finished

```bash
cd weather
npm start
```

## Building an application

```bash
npm run build
```
